Module Uuseg_gen.Grapheme_cluster

This module includes some convenience functions for working with grapheme cluster boundaries. These generators produce `Uchar c if c is not the last character of a grapheme cluster and produce `LastUchar c if c is the last character of a grapheme cluster.
